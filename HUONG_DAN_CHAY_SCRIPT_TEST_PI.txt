═══════════════════════════════════════════════════════════════
NẾU GẶP LỖI:
═══════════════════════════════════════════════════════════════

1. Lỗi "Module not found":
   → Cài đặt: pip3 install requests urllib3
   → Cài OpenCV: sudo apt install python3-opencv

2. Lỗi "Serial number not found":
   → Tạo file serial_number.txt với Serial number thật
   → Hoặc sửa trong script: SERIAL_NUMBER = "202500000"

3. Lỗi "Client not found":
   → Kiểm tra Serial number trên Pi và server có khớp không
   → Tạo client trên server với Serial number đúng

4. Lỗi kết nối:
   → Kiểm tra internet
   → Kiểm tra firewall
   → Kiểm tra server có đang chạy không
